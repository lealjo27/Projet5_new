# ✨ Fonctionnalités

Le projet propose une API complète pour la prédiction d'attrition des employés.

---

## Fonctionnalités principales

- ✅ Prédiction d'attrition en temps réel avec probabilité
- ✅ Authentification sécurisée par JWT
- ✅ API REST documentée automatiquement avec Swagger/OpenAPI
- ✅ Historique des prédictions sauvegardé en base de données
- ✅ Logging des appels API
- ✅ Tests unitaires avec `pytest`
- ✅ Gestion d'erreurs claire et robuste
- ✅ Conteneurisation avec Docker
- ✅ Persistance des données avec NeonDB PostgreSQL

---

## Fonctionnalités API

L'API permet notamment de :

- vérifier que le service est actif ;
- obtenir un token JWT ;
- effectuer une prédiction pour un employé existant ;
- obtenir une réponse claire en cas d'erreur ;
- sauvegarder les prédictions réalisées ;
- tracer les temps d'exécution des appels.
